Preguntas de la actividad 1



¿Qué son los componentes pasivos en la red?

Un componente pasivo, o dispositivo pasivo, es un componente electrónico que solo puede recibir energía, que puede disipar, absorber o almacenar en un campo eléctrico o magnético. Los componentes pasivos no necesitan ningún tipo de energía eléctrica independiente para funcionar.

Ejemplos de componentes electrónicos pasivos son condensadores, resistencias, inductores, transformadores y diodos.3





RJ45

El cableado RJ45 también se conoce como cable de par trenzado para Ethernet (TPE), cable par trenzado sin blindaje (UPT) y cableado 10BASE-T. El tipo de cableado que debe usar depende de su adaptador.

Keystone

Un conector Keystone es un conector hembra utilizado en las comunicaciones de datos, en particular para las redes de área local (LAN), normalmente montado en una placa de pared o panel de conexiones.

Cable UTP

Los cables UTP se utilizan para la creación de redes, señales de voz, audio, videos o de datos. UTP, acrónimo inglés de Unshielded Twisted Pair, o par trenzado sin apantallar, es un tipo de cable que se utiliza en las telecomunicaciones y redes informáticas.



Patch panel

Un patch panel es un dispositivo de red eficaz y flexible para mantener organizado tu centro de datos o sala de servidores, así como para facilitar el traslado, la adición o el cambio de la infraestructura de cableado en el futuro.



Patch cord

El Cable de Parcheo sirve entonces para conectar esa salida del Panel hacia otro equipo que podría ser un Switch o podría ser incluso otro Panel de Parcheo en el caso de un Cross Connect.



Fibra óptica

La fibra óptica es una guía de onda en forma de hilo de material altamente transparente diseñado para transmitir información a grandes distancias utilizando señales ópticas.



Canaletas

Las canaletas para cables son tubos aplanados que se fijan en las paredes, pisos o techos para transportar los cables de un circuito, de modo que llevan electricidad de forma segura y efectiva.




¿Qué son los componentes activos en la red?

Los elementos activos son aquellos que pueden controlar el flujo de electricidad. La mayoría de las placas de circuito impreso tienen al menos un componente activo. Algunos ejemplos de componentes electrónicos activos son transistores, tubos de vacío, rectificadores controlados de silicio.





Switch

Un switch o conmutador es un dispositivo de interconexión utilizado para conectar equipos en red formando lo que se conoce como una red de área local (LAN) y cuyas especificaciones técnicas siguen el estándar conocido como Ethernet (o técnicamente IEEE 802.3).

Router

Un router es un dispositivo que proporciona Wi-Fi y que generalmente está conectado a un módem. Envía información desde Internet a los dispositivos personales, como computadoras, teléfonos o tablets.

Tarjeta de red

Una tarjeta de red es un dispositivo que se instala en el interior de un ordenador para que este pueda conectarse a Internet. Según el tipo de tarjeta, esta puede incluir en su parte posterior un puerto para introducir el cable de Ethernet o admitir la conexión inalámbrica.



Interruptor inteligente wifi

Los interruptores inteligentes vinculados al wifi son dispositivos que domotizan la casa invirtiendo muy pocos recursos: ofrecen la misma funcionalidad que un interruptor clásico pero gestionan de manera automática la iluminación y el funcionamiento de los electrodomésticos, tomando el control desde el móvil



Access Point

El Access Point entonces se encarga de ser una puerta de entrada a la red inalámbrica en un lugar específico y para una cobertura de radio determinada, para cualquier dispositivo que solicite acceder, siempre y cuando esté configurado y tenga los permisos necesarios.




Herramientas utilizadas para realizar cableado estructurado


Ponchadora

Una ponchadora RJ 45 es una herramienta que se compone de unas pinzas que ejercen una gran presión y sirven para presionar fuertemente empalmes para los cables eléctricos o zapatas eléctrica.

Crimpadora

Una crimpadora cortadora de cable es una herramienta que se utiliza para cortar y unir terminales metálicos o conectores a los extremos de los cables eléctricos. Una crimpadora eléctrica tiene dos funciones principales, el corte de cables y el crimpado de terminales. Corte de cables.

Desforradora

Esta herramienta está diseñada para remover de manera rápida segura precisa chaquetas de cable y alambre de puesta a tierra. Excelente herramienta para remover con gran precisión chaquetas de cables multifibra.

Tester

El tester es una herramienta de medición que se puede usar en varios oficios, básicamente mide resistencia, tensión, intensidad, y continuidad, pero existen multímetros en el mercado que sirven para otras mediciones, como frecuencia, audio, luz, temperatura, etc.



¿Qué es el cableado horizontal?

La norma EIA/TIA 568A define el cableado horizontal de la siguiente forma: El sistema de cableado horizontal es la porción del sistema de cableado de telecomunicaciones que se extiende del área de trabajo al cuarto de telecomunicaciones o viceversa.

¿Qué es el cableado vertical?

El propósito del cableado del backbone es proporcionar interconexiones entre cuartos de entrada de servicios de edificio, cuartos de equipo y cuartos de telecomunicaciones. El cableado del backbone incluye la conexión vertical entre pisos en edificios de varios pisos.




Tipos de cables para el sistema de cableado horizontal

Cables de Par Trenzado Sin Blindar (UTP) de 100 Ohm y Cuatro Pares:
El cable UTP (Unshielded Twisted Pair) es uno de los tipos de cables más comunes utilizados en redes de telecomunicaciones. Está compuesto por cuatro pares de hilos de cobre trenzados entre sí, lo que ayuda a reducir la interferencia electromagnética y la diafonía entre los pares. No tiene blindaje adicional, lo que lo hace más flexible y fácil de instalar.
Aplicaciones: Es ampliamente utilizado en redes Ethernet y telefonía, principalmente para conexiones de red de corta y media distancia.



Cables de Par Trenzado Blindados (STP) de 150 Ohm y Cuatro Pares:
Descripción: El cable STP (Shielded Twisted Pair) es similar al UTP, pero cuenta con una capa de blindaje adicional alrededor de los pares de cables trenzados. Este blindaje puede ser una lámina o malla de metal que ayuda a proteger contra interferencias electromagnéticas externas y reducir la diafonía.
Aplicaciones: Es utilizado en entornos donde hay una alta interferencia electromagnética o se requiere una mayor protección para las señales de red, como en instalaciones industriales o de alta seguridad.



Cables de Fibra Óptica Multimodo de 62.5/125 µm y Dos Fibras:
Descripción: Este tipo de cable utiliza fibras ópticas para transmitir datos a través de pulsos de luz. La fibra óptica multimodo de 62.5/125 µm se refiere a un núcleo de 62.5 micrómetros de diámetro rodeado por un revestimiento de 125 micrómetros. La fibra óptica multimodo permite la transmisión de múltiples modos o caminos de luz, lo que es adecuado para distancias cortas y medianas.
Aplicaciones: Es ideal para redes de alta velocidad y largo alcance en aplicaciones de transmisión de datos, como conexiones entre edificios o dentro de un centro de datos, donde se requieren grandes anchos de banda y bajas interferencias.





Normativa para hacer cables de red (clase A y B)





Para que se utiliza un cable directo en la red?


El cable directo de red sirve para conectar dispositivos desiguales, como un computador con un hub o switch. En este caso, ambos extremos del cable deben tener la misma distribución.

Para que se utiliza un cable cruzado en la red?


En general, un cable cruzado se utiliza para conectar dos dispositivos del mismo tipo, como por ejemplo un PC a una PC o un switch a otro switch. Por otro lado, el cable directo conecta dos dispositivos diferentes entre sí, como por ejemplo un PC y un switch
